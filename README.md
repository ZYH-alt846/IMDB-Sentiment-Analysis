# IMDB影评情感分析（LSTM+注意力机制）
本项目基于LSTM+注意力机制，结合GloVe预训练词向量，完成IMDB电影评论的情感二分类（正面/负面）。


## 一、项目流程
1. **数据预处理**：清洗影评文本→ 单词转数字索引 → 统一文本长度为512；
2. **词向量加载**：使用GloVe 840B 300维预训练词向量；
3. **模型训练**：搭建LSTM+注意力机制模型，训练10轮；
4. **结果预测**：对25000条测试集影评做情感预测，输出CSV结果文件。


## 二、核心训练数据
| 训练指标       | 最终数值 |
|----------------|----------|
| 训练轮数       | 10轮     |
| 训练集准确率   | 【90%】  | 
| 验证集准确率   | 【86%】  |
| 单轮训练耗时   | 25-27秒  |
| 测试集样本总数 | 25000条  |


## 三、测试集预测结果（`attention_lstm (1).csv`）
- 正面情感（标签1）：【11297】条 
- 负面情感（标签0）：【13703】条 
- 文件格式：包含影评唯一标识（id）」和 情感标签（sentiment）
